### 📝 Project Overview
This application is a React-based dashboard that simulates a sports auction environment. Users can claim free credits to start their journey and spend them wisely on top-tier players. The project focuses on efficient state management, preventing over-budgeting, and ensuring a seamless user experience through interactive UI components.

### ✨ Key Features
- **💰 Virtual Credit System:** Users can claim free "coins" and use them to purchase players. The balance updates dynamically.
- **🔄 Smart View Toggle:** Easily switch between the **Available Players** market and the **Selected Squad** list.
- **🛡️ Validation Logic:** - Prevents selecting the same player twice.
  - Enforces a maximum limit of 6 players per squad.
  - Checks for "Insufficient Balance" before allowing a purchase.
- **📱 Responsive Design:** Built with **Tailwind CSS** and **DaisyUI** for a mobile-first, professional aesthetic.
- **🔔 Real-time Notifications:** Instant feedback when a player is added, removed, or if a selection is invalid.

### 💻 Tech Stack
- **Frontend:** React.js
- **Styling:** Tailwind CSS, DaisyUI
- **Icons:** React Icons
- **State Management:** Functional Components & Hooks (useState, useEffect)
